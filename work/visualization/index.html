Visualizations of Real time credit-card Transactions

Role | Interaction Designer, Front-end Developer

Technology Stack | HTML5, CSS3, D3js, jQuery, Javascript, Web Sockets, NodeJS

As part of a demo for a multi-national financial services corporation, we were supposed to develop, a Proof of Concept on visualizing number of real-time transactions in parts of United States over the map.

My responsibilities included working on the front-end interface and the visualization using D3.js. This POC was a great learning experience as it gave me an opportunity to work in the area of Data Visualization, for which I had developed a recent interest and got to code the same using D3.
